System: unicycle_static_obstacle
Description: Unicycle robot navigating around a circular static obstacle. The CBF ensures the robot maintains a safe distance h(x) >= 0 from the obstacle throughout the trajectory.

Controllers:
  Legacy: LQR / pure pursuit (nominal, no CBF)
  Evolved: CBF-QP with barrier constraint (robust safety filter)
  CBF type: Distance-based barrier: h(x) = ||p - p_obs||^2 - r^2

Simulation:
  Time step: 0.07 s
  Duration: 20.0 s
  Samples: 200
  Safety condition: h(x(t)) = ||position - obstacle_center||^2 - obstacle_radius^2 >= 0
  Seeds: Uniform random sampling over input ranges, shared seeds between legacy and evolved

Input Features (4):
  initial_distance_to_obstacle: Euclidean distance from unicycle to static obstacle at t=0 [m] sampled [0.5, 5.0], physical [0.0, 100.0]
  initial_speed: Initial linear speed of the unicycle [m/s] sampled [0.1, 3.0], physical [0.0, 10.0]
  obstacle_radius: Radius of the circular static obstacle [m] sampled [0.2, 1.5], physical [0.01, 5.0]
  initial_heading_error: Difference between unicycle heading and goal direction at t=0 [rad] sampled [-3.14, 3.14], physical [-3.14159, 3.14159]
